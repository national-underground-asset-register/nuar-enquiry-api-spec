Proposed Works Enquiry API (1.0.0)

Download OpenAPI specification:

License: Open Government Licence

A specification for an API that allows details of proposed works at various project phases to be received in order for asset owners (or their service providers) to receive notification of potential works in order to carry out risk assessments and prioritise operational activities to reduce the risk of damage to assets

Submit a new enquiry

Allow a new enquiry to be submitted from the source application

Authorizations:
BasicAuth
Request Body schema: application/json
required
sourceEnquiryID
string <uuid> (Enquiry ID)

The Unique identifer of the enquiry in the source system

projectName
string or null <= 250 characters

The name of any project of which the enquiry is part

enquiryTitle
required
string <= 255 characters

The name or title of this enquiry

enquiryReference
required
string <= 100 characters

The reference of this enquiry

enquiryPhase
required
string (EnquiryPhase)
Enum: "Information only" "Design or feasibility" "Planned works" "Reactive works"

An enumeration of possible project phases for which the enquiry is being made

isEmergency
boolean
Default: false

An indication of whether the works in question are emergency works necessitating a fast response

required
object (EnquirerDetails)
enquiryDescription
required
string

A description of the project and the works being planned along with any other pertinent descriptive information

additionalInformation
string or null

Any additional information which may be of use to those assessing the works and their impact on their assets. This may include: onsite contact details, details and dimensions of any structures being constructed, details of site ingress and egress which may involve heavy plant etc.

enquiryLocation
required
object

A GeoJSON geometry object representing the location, extent or route of the proposed works

locationGeometryCRS
string
Default: "27700"

The coordinate reference system of the geometry used to specify the work location. Specified as the numeric part of the EPSG reference

dateTimeCreated
required
string <date-time>

The date and time on which the enquiry was created

activityStartDate
required
string <date-time>

The start date for any proposed works. This may be an estimate if the planning is in its early stages

activityEndDate
string or null <date-time>
activityType
required
string (ActivityType)
Enum: "Bank works" "Blasting" "Building construction" "Cranes / work platforms" "Agricultural groundworks" "Demolition (explosives)" "Demolition (no explosives)" "Ditch works / Field drainage" "Dredging" "Fencing" "Field drainage" "Gulley works" "Horizontal drilling / moling / tunnelling / pipe jacking" "Hot works" "Installation of buried services" "Landfilling" "Landscaping" "Mast site" "Multiple excavations site (1.5m or shallower)" "Multiple excavations site (deeper than 1.5m)" "Other" "Piling" "Pole works" "Preliminary Enquiry" "Pressure testing" "Private services" "Resurfacing" "Road / track construction" "Seismic survey" "Single excavation site (1.5m or shallower)" "Single excavation site (deeper than 1.5m)" "Site survey (non-invasive)" "Solar farm" "Street furniture" "Traffic calming" "Tree planting" "Tree removal" "Trial holes" "Wind farm" "Works to alter the ground level" "Works to alter the water level"

An enumeration of possible activity types which may be planned or proposed

Array of objects (Rule)

Details of the rules that have been triggered in order for this notification to be raised
